# SCT_AD_2 — Simple To-Do List App

## 📌 Project Title
**SCT_AD_2 - Simple To-Do List App**

## 📖 Description
A beginner-friendly Android application built using **Kotlin** and **XML** that allows users
to manage a simple to-do list. Tasks are stored in memory (no database) and are displayed
using a `ListView`. Users can add new tasks and delete existing ones with a long press.

This project was developed as part of an Android Development internship task.

## ✨ Features
- ✅ Add new tasks to the list
- ✅ Delete a task by long-pressing on it
- ✅ Toast message shown when a task is added
- ✅ Toast message shown when a task is deleted
- ✅ Empty input validation (prevents adding blank tasks)
- ✅ Clean, modern, and attractive Material Design UI
- ✅ Rounded Material Buttons and Input Fields

## 🛠️ Tech Stack
- **Language:** Kotlin
- **UI:** XML with Material Design 3 Components
- **Architecture:** Single Activity (no MVVM)
- **View Binding:** Enabled
- **Data Storage:** In-memory (`ArrayList<String>`) — no Room/Firebase
- **List Display:** `ListView` with `ArrayAdapter`
- **Min SDK:** 24
- **Target SDK:** 34
- **Compile SDK:** 34

## 🚀 How to Run
1. Clone or download this repository.
2. Open **Android Studio**.
3. Select **Open an Existing Project** and choose the `SCT_AD_2` folder.
4. Let Gradle sync complete.
5. Connect an emulator or physical device (Android 7.0 / API 24 or higher).
6. Click **Run ▶️** to build and launch the app.

## 📱 How to Use
1. Type a task into the input field.
2. Tap **Add Task** to add it to your list.
3. Long-press any task in the list to delete it.
